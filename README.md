### How to install
```
python setup.py bdist_wheel
pip install --upgrade dist/visualdl-0.0.1-py2-none-any.whl
```

### Run a demo from scratch
```
vdl_scratch.py
visualDL --logdir=scratch_log
```
that will start a server locally.
